## 앱 정보: 루티팜 (RoutineFarm)

### 컨셉
매일 1분짜리 건강 미션을 완료하면 다마고치 캐릭터가 성장하는 미니앱.
습관을 키우고 기른다는 의미를 담은 앱.

### 타겟 사용자
불특정 다수

### 카테고리
스트레칭/운동, 물 마시기/식습관, 눈/목 피로 해소, 명상/호흡, 정리정돈

### 미션 방식
사용자가 직접 미션 추가 가능

### 캐릭터 성장 시스템
| 스트릭 | 상태 |
|---|---|
| 0일 | 🥚 알 |
| 3일 | 🐣 부화 |
| 7일 | 🐥 병아리 |
| 14일 | 🐔 진화 |
| 하루 빠지면 | 😢 배고픈 상태 |
| 3일 방치 | 😵 기절 상태 |

### 수익 모델
인앱 광고 (보상형) - 광고 보면 캐릭터 밥 주기 가능
