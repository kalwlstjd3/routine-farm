# 루티팜 테스트 시나리오

## 시나리오 1: 온보딩
1. localStorage.clear() 후 새로고침
2. 온보딩 화면 표시 확인
3. "시작하기" 클릭 → 홈화면 이동 확인
4. 홈화면에 🥚 회색 알 + "첫 미션을 완료하면 새로운 친구를 만날 수 있어요!" 문구 확인

## 시나리오 2: 첫 미션 완료 → 펫 뽑기
1. 미션 고르기 → 미션 선택 → 미션 시작하기
2. 타이머 60초 완료 후 "완료하기" 활성화 확인
3. "완료하기" 클릭 → PetGachaPage 이동 확인
4. "두근두근..." 문구 + 2초 로딩 후 랜덤 펫 공개 확인
5. "다시 뽑기" 클릭 → 다른 펫으로 교체 확인
6. "이 아이로 할게요!" 클릭 → 홈화면 펫 표시 확인

## 시나리오 3: 미션 완료 & 연속 달성 일수
1. 미션 완료 후 연속 달성 일수 1일 확인
2. 버튼 "오늘 미션 완료! ✓" 비활성화 확인
3. 새로고침 후에도 데이터 유지 확인

## 시나리오 4: 캐릭터 성장 (콘솔에서 테스트)
아래 값을 브라우저 콘솔에서 입력 후 새로고침으로 확인
const, let 대신 var 사용 (콘솔 재선언 에러 방지)

⚠️ 주의: 시나리오 2(펫 뽑기)를 먼저 완료하거나, 아래 사전 설정을 실행해야 함
배고파요/기절은 my_pet이 설정된 경우에만 UI에 표시됨
last_fed가 오늘로 남아 있으면 배고파요 상태가 덮어씌워지므로 반드시 삭제

// 0단계 - 사전 설정 (펫 없거나 last_fed 잔존 시 먼저 실행)
localStorage.setItem('my_pet', 'chicken')
localStorage.removeItem('last_fed')

// 1단계 - 먼저 실행 (헬퍼 선언)
var today = new Date().toISOString().slice(0, 10)
var yesterday = new Date(Date.now() - 86400000).toISOString().slice(0, 10)
var threeDaysAgo = new Date(Date.now() - 86400000 * 3).toISOString().slice(0, 10)

// 3일 → 부화중/조금 자람 (새로고침)
localStorage.setItem('streak', '3')
localStorage.setItem('mission_done', today)

// 7일 → 새끼/성체 (새로고침)
localStorage.setItem('streak', '7')
localStorage.setItem('mission_done', today)

// 14일 → 완전 성체 (새로고침, 팝업은 미션 완료 시에만 뜸 → 시나리오 5 참고)
localStorage.setItem('streak', '14')
localStorage.setItem('mission_done', today)

// 방치 1일 → 배고파요 (새로고침)
localStorage.removeItem('last_fed')
localStorage.setItem('streak', '3')
localStorage.setItem('mission_done', yesterday)

// 방치 3일+ → 기절 (새로고침)
localStorage.removeItem('last_fed')
localStorage.setItem('streak', '3')
localStorage.setItem('mission_done', threeDaysAgo)

## 시나리오 5: 14일 달성 팝업
⚠️ 팝업은 completeMission()이 호출될 때만 표시됨 (localStorage 직접 수정으로는 재현 불가)
실제 미션을 완료해서 streak가 14가 되는 시점에 팝업이 뜨는지 확인해야 함

콘솔 단축 재현 방법:
1. localStorage.setItem('streak', '13') + localStorage.setItem('mission_done', yesterday) 설정 후 새로고침
2. 미션 선택 → 타이머 60초 완료 → "완료하기" 클릭 → "🎉 완전 성체 달성!" 팝업 확인
3. "이 아이와 계속하기" → 팝업 닫히고 현재 펫 유지 확인
4. "새 펫 뽑기 (광고 보기)" → PetGachaPage 이동 확인

## 시나리오 6: 미션 선택
1. 홈 "미션 고르기 →" 클릭 → MissionSelectPage 이동
2. 카테고리 탭 5개 전환 확인
3. 🎲 랜덤 버튼 → 랜덤 미션 자동 선택 확인
4. 미션 선택 후 "이 미션 선택" → 홈 복귀 + 미션 표시 확인
5. 새로고침 후에도 선택한 미션 유지 확인
6. 다음날 시뮬레이션 (콘솔):
localStorage.setItem('today_mission', JSON.stringify({date: '2026-04-01', mission: {}}))
새로고침 후 미션 초기화 확인

## 시나리오 7: 밥 주기 (방치 상태)
1. 시나리오 4에서 방치 상태 만들기
2. 홈화면에 "광고 보고 캐릭터 밥 주기" 버튼 확인
3. 버튼 클릭 → 캐릭터 상태 회복 확인
